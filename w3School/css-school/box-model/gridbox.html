The Dispaly: inline-block Value

display: inline

Compare to display: inline, the major difference is that display: inline-block allows to set a width and height on the element

Also, with display: inline-block, the top and bottom margin/padings are respected, but with display: inline they are not

Compared display: block, the major difference is that display: inline-block does not add a line break afte the element, so the element can sit next to other elements.
