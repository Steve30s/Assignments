What is boxsing

You can easily create floatng boxes side by side, when you add somethig that enlarges the width of each box(e.g padding or borders), the box will break. Theboxsizng property allows us to include the padin and borfer in the box's total width ( and height), making sure that the padding stays inside of the box and that it does not break
